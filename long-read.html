長讀長定序分析 (Long-read)

PacBio HiFi 與 ONT R10.4.1：解開 short-read 的盲區、SV、phasing 與表觀基因組。

PacBio HiFi and ONT R10.4.1: unlock short-read blind spots, SVs, phasing, and epigenome.

概覽

為什麼需要 Long-read？

Short-read 在 (1) 重複區域 (segmental duplications、tandem repeats、centromere)、(2) 大型 SV、(3) haplotype phasing、(4) 5mC/6mA 表觀修飾 都有先天限制。Long-read 一條讀就可橫跨數 kb–Mb，直接解這些問題。

2026 兩大平台：(1) PacBio HiFi（CCS reads，Q30+ 共識準確）— Revio 平台高通量、~15–25 kb 讀長；(2) ONT R10.4.1 + Dorado basecaller — raw ~99.5%，duplex 模式可達 HiFi 等級，讀長可 > 100 kb。

Short reads struggle with (1) repeat regions (segdup, tandem repeats, centromere), (2) large SVs, (3) haplotype phasing, (4) 5mC/6mA epigenome. One long read spans kb–Mb and resolves these directly.

Two 2026 platforms: (1) PacBio HiFi (CCS reads, Q30+ consensus) — Revio platform, ~15–25 kb; (2) ONT R10.4.1 + Dorado basecaller — raw ~99.5%, duplex matches HiFi, reads > 100 kb.

💡
T2T-CHM13 v2.0：Nurk 2022 (Science) 發表首個真正 telomere-to-telomere 完整人類基因組——包含過去 GRCh38 缺失的 ~200 Mb 重複序列。Long-read 是這項里程碑的關鍵。 T2T-CHM13 v2.0: Nurk 2022 (Science) — first true telomere-to-telomere human genome, adding ~200 Mb of repeat regions missing from GRCh38. Long-read made this milestone possible.

二、HiFi vs ONT：怎麼選？

🌳 專案需求 → 平台

Q1:
需要高準確 SNV/Indel calling？ → HiFi（Q30+，等同 short-read 準度）。
Q2:
需要 > 100 kb 超長 read 解 segmental duplication？ → ONT ultra-long（可達數 Mb）。
Q3:
需要即時/床邊定序、低初期成本？ → ONT MinION/PromethION。
Q4:
需 RNA isoform 同時看甲基化？ → 兩者皆可（PacBio Iso-Seq / ONT direct RNA）。
Q1:
High-accuracy SNV/Indel? → HiFi (Q30+, matches short-read).
Q2:
Need >100 kb to resolve segdup? → ONT ultra-long (multi-Mb possible).
Q3:
Real-time / bedside / low upfront cost? → ONT MinION/PromethION.
Q4:
RNA isoforms + methylation? → Both (PacBio Iso-Seq / ONT direct RNA).

程式碼

Long-read 三步驟

# minimap2 — HiFi vs ONT presets
minimap2 -ax map-hifi -t 16 ref.fa hifi.fq.gz | samtools sort -@8 -o hifi.bam
minimap2 -ax map-ont  -t 16 ref.fa ont.fq.gz  | samtools sort -@8 -o ont.bam
samtools index hifi.bam && samtools index ont.bam

# T2T-CHM13 v2.0 — use as reference for full genome coverage
wget https://s3-us-west-2.amazonaws.com/human-pangenomics/T2T/CHM13/assemblies/chm13v2.0.fa.gz
# Clair3 — small variant caller for ONT & HiFi
run_clair3.sh \
    --bam_fn=ont.bam \
    --ref_fn=ref.fa \
    --threads=16 \
    --platform=ont \
    --model_path=/clair3_models/r1041_e82_400bps_sup_v500 \
    --output=clair3_out

# DeepVariant for HiFi
run_deepvariant --model_type=PACBIO --ref=ref.fa \
    --reads=hifi.bam --output_vcf=hifi.vcf.gz
# Sniffles2 — SV calling for long reads
sniffles \
    --input ont.bam \
    --reference ref.fa \
    --vcf sv.vcf.gz \
    --tandem-repeats human_GRCh38_no_alt_analysis_set.trf.bed \
    --threads 8

# Phasing with WhatsHap
whatshap phase -o phased.vcf --reference ref.fa snv.vcf.gz hifi.bam

# Methylation directly from ONT raw signal
dorado basecaller sup --modified-bases 5mCG_5hmCG pod5/ > mod.bam
