STEP 10 / 16

變異註釋 (VEP / SnpEff)

把座標變成「臨床意義」：轉錄本選擇、功能預測、族群頻率與深度學習評分整合。

From coordinates to clinical meaning: transcript choice, functional prediction, population frequency, and deep-learning scores combined.

概覽

變異註釋要回答的問題

一個 chr7:140,453,136 A>T 的座標對臨床無意義，必須回答：(1) 落在哪個基因 / 轉錄本？(2) coding / non-coding / splice / UTR？(3) 改變胺基酸嗎？(4) 族群頻率多少？(5) 致病性預測？(6) 是否有 ClinVar / OMIM 紀錄？這就是「註釋」要做的事。

關鍵挑戰：同一個位置可能屬於多個轉錄本（人類平均每基因 5–10 條 transcript），導致註釋輸出不一致。2022 年 NCBI 與 Ensembl 共同發布 MANE Select (Morales 2022, Nature)，為每個 protein-coding gene 指定唯一「最具代表性」的轉錄本，已成臨床標準。

A bare coordinate (chr7:140,453,136 A>T) has no clinical meaning. Annotation answers: (1) which gene/transcript? (2) coding / non-coding / splice / UTR? (3) amino-acid change? (4) population AF? (5) pathogenicity prediction? (6) ClinVar/OMIM record?

Key challenge: one site can map to multiple transcripts (5–10 per gene on average), making annotation inconsistent. In 2022 NCBI + Ensembl jointly released MANE Select (Morales 2022, Nature) — one canonical transcript per protein-coding gene — now the clinical standard.

💡
核心三件套：(1) 註釋器 (VEP 或 SnpEff)　(2) 族群頻率 (gnomAD v4)　(3) 致病性預測 (CADD / REVEL / AlphaMissense / SpliceAI)。透過 dbNSFP 一次取得 40+ 種預測分數。 The core trio: (1) annotator (VEP or SnpEff) (2) population AF (gnomAD v4) (3) pathogenicity (CADD / REVEL / AlphaMissense / SpliceAI) — dbNSFP bundles 40+ scores at once.

致病性預測

二、Missense 與 Splice 預測工具

🔬

CADD

Kircher 2014, Nat Genet。整合 60+ 注釋，Phred-scaled，全基因組可用。CADD ≥ 20 ≈ 前 1% 有害變異。

Kircher 2014. Integrates 60+ annotations; Phred-scaled; genome-wide. CADD ≥ 20 ≈ top 1% damaging.

🎯

REVEL

Ioannidis 2016, AJHG。Random Forest 整合 13 個 missense 預測，分數 0-1。> 0.5 一般視為致病傾向。

Ioannidis 2016. Random forest of 13 missense predictors, score 0–1. > 0.5 generally pathogenic-leaning.

🧠

AlphaMissense

Cheng 2023, Science。DeepMind 用 AlphaFold 結構預測 missense 致病性，分類為 likely_benign / ambiguous / likely_pathogenic。

Cheng 2023, Science. DeepMind uses AlphaFold structure to classify missense as likely_benign / ambiguous / likely_pathogenic.

✂️

SpliceAI

Jaganathan 2019, Cell。Illumina 開發的深度學習 splice 預測，輸出 4 個分數（acceptor/donor × gain/loss）。> 0.2 視為可能影響 splicing。

Jaganathan 2019, Cell. Deep-learning splice predictor by Illumina; 4 scores (acceptor/donor × gain/loss). > 0.2 = potentially splice-affecting.

📚

dbNSFP 4.x

Liu 2020, Genome Med。預先計算所有可能 missense 變異的 40+ 種預測分數，VEP 可一次取得。

Liu 2020. Precomputes 40+ predictor scores for every possible missense; VEP loads in one shot.

🗂️

ClinVar

NIH 公開致病性資料庫。VEP 透過 --custom 即可整合 Pathogenic / Likely Pathogenic / VUS / Benign 五級分類。

NIH public clinical variant database; VEP loads via --custom with five-level pathogenicity tiers.

⚠️
不要盲信單一分數：AlphaMissense 也會錯（特別是結構未解析區域）。臨床上要結合 ACMG/AMP 規則、ClinVar、家系分離與功能實驗——分數只是 supporting evidence，不能單獨定性。 Never trust a single score: AlphaMissense can be wrong (especially in poorly resolved regions). Clinically, combine ACMG/AMP rules, ClinVar, segregation, and functional assays — scores are supporting evidence only.

典型 WES 一個樣本約 ~25,000 個 coding 變異。拖動 AlphaMissense 與 gnomAD AF 閾值，觀察候選變異如何「漏斗收斂」。

A typical WES sample yields ~25,000 coding variants. Drag AlphaMissense and gnomAD AF cutoffs to watch the candidate funnel narrow.

程式碼

實作範例

# VEP — 全套臨床註釋（MANE + dbNSFP + SpliceAI + AlphaMissense + gnomAD）
vep -i joint/cohort.filt.vcf.gz \
    -o annot/cohort.vep.vcf.gz --vcf --compress_output bgzip \
    --species homo_sapiens --assembly GRCh38 \
    --cache --dir_cache $VEP_CACHE --offline \
    --mane --pick_allele_gene \
    --plugin dbNSFP,dbNSFP4.7a.gz,REVEL_score,CADD_phred,AlphaMissense_score \
    --plugin SpliceAI,snv=spliceai.snv.vcf.gz,indel=spliceai.indel.vcf.gz \
    --custom gnomad.v4.exomes.vcf.gz,gnomADe,vcf,exact,0,AF,AF_popmax \
    --custom clinvar.vcf.gz,ClinVar,vcf,exact,0,CLNSIG,CLNREVSTAT \
    --fork 8
# SnpEff — 快速 (適合非人類 / 自建 DB)
snpEff -v GRCh38.105 joint/cohort.filt.vcf.gz > annot/cohort.snpeff.vcf

# 自建非模式生物 DB
mkdir -p $SNPEFF_DATA/myorganism && \
    cp genes.gbk $SNPEFF_DATA/myorganism/
snpEff build -genbank -v myorganism

# 補上 dbNSFP 分數
SnpSift dbnsfp -v -db dbNSFP4.7a.gz -f REVEL_score,CADD_phred annot/cohort.snpeff.vcf \
    > annot/cohort.snpeff.dbnsfp.vcf
# 漏斗篩選：罕見 + 致病傾向 + ClinVar Pathogenic
bcftools view -i 'INFO/gnomADe_AF<0.001' annot/cohort.vep.vcf.gz \
  | bcftools view -i 'INFO/CSQ ~ "HIGH" || INFO/CSQ ~ "MODERATE"' \
  | bcftools view -i 'INFO/CSQ ~ "AlphaMissense_score=0\.[5-9]"' \
  | bgzip > candidates.vcf.gz

# 報表表格
bcftools +split-vep candidates.vcf.gz -f '%CHROM\t%POS\t%REF\t%ALT\t%SYMBOL\t%Consequence\t%AlphaMissense_score\t%gnomADe_AF\n' \
    -d -A tab > candidates.tsv
