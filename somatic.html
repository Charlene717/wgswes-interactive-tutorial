STEP 12 / 16

體細胞變異呼叫 (Somatic Calling)

從腫瘤組織中找出後天突變——低 VAF、purity、subclonality、FFPE artefact 與 ctDNA 偵測極限。

Find acquired mutations in tumor tissue — low VAF, purity, subclonality, FFPE artefacts, and ctDNA detection limits.

概覽

Somatic 與 Germline 的根本差異

Germline 變異期望 VAF 在 50% (het) 或 100% (hom)；Somatic 變異 VAF 可低至 1% 甚至 0.01% (ctDNA)，且每個 subclone 的 VAF 不同。Caller 不能假設 diploid 模型，必須處理 (1) 腫瘤純度、(2) 拷貝數變異、(3) FFPE 化學損傷、(4) sequencing artefact。

主流方案：GATK Mutect2 (Benjamin 2019, BioRxiv) — 採用 PairHMM + Bayesian model；Strelka2 (Kim 2018, Nat Methods) — 速度快、準確度與 Mutect2 相當；VarScan2、DeepSomatic (Google 2024)、Lancet、SAGE 為輔助。

Germline variants expect VAF ~50% (het) or 100% (hom). Somatic variants can be VAF 1% or even 0.01% (ctDNA), with different VAFs per subclone. Callers must handle (1) tumor purity, (2) CNV, (3) FFPE damage, (4) sequencing artefacts — they cannot assume diploid.

Mainstream: GATK Mutect2 (Benjamin 2019, BioRxiv) with PairHMM + Bayesian model; Strelka2 (Kim 2018, Nat Methods) — fast, comparable accuracy. VarScan2, DeepSomatic (Google 2024), Lancet, SAGE are auxiliary.

💡
兩種模式：(1) Tumor-Normal paired — 同一個體的腫瘤 + 對照血液，可直接扣除 germline；(2) Tumor-only — 只有腫瘤，須用 Panel of Normals (PoN) + gnomAD 過濾 germline。Paired 永遠較準。 Two modes: (1) Tumor-Normal paired — tumor + matched normal from same individual, germline subtracted directly; (2) Tumor-only — tumor alone, must use Panel of Normals (PoN) + gnomAD to filter germline. Paired is always more accurate.

決策引導

二、Subclonality、ctDNA 與 MRD

🌳 從 VAF 到臨床判讀

Q1:
VAF ~ 0.5 × purity？ → 是 → Clonal heterozygous（早期 driver）。
Q2:
VAF < 0.5 × purity 但 > 5%？ → 是 → Subclonal，用 PyClone-VI (Gillis 2020) 或 sciClone 估計 subclone 結構。
Q3:
VAF < 1% 且來自血漿 cfDNA？ → 是 → ctDNA / MRD：需 UMI consensus + 超深定序 (≥ 30,000×)。
Q4:
單一 read 支持 + 無 UMI？ → 是 → 視為 sequencing artefact，丟棄。
Q1:
VAF ~ 0.5 × purity? → Yes → Clonal heterozygous (early driver).
Q2:
VAF < 0.5 × purity but > 5%? → Yes → Subclonal; use PyClone-VI (Gillis 2020) or sciClone for subclone structure.
Q3:
VAF < 1% from plasma cfDNA? → Yes → ctDNA / MRD: requires UMI consensus + ultra-deep (≥ 30,000×).
Q4:
Single-read support + no UMI? → Yes → Treat as sequencing artefact, discard.

程式碼

Mutect2 + 過濾流程

# Mutect2 tumor-normal paired with PoN + gnomAD AF resource
gatk Mutect2 \
    -R reference.fa \
    -I tumor.bam   -tumor  TUMOR_SM \
    -I normal.bam  -normal NORMAL_SM \
    --germline-resource af-only-gnomad.hg38.vcf.gz \
    --panel-of-normals pon.vcf.gz \
    --f1r2-tar-gz f1r2.tar.gz \
    -L exome.bed \
    -O unfiltered.vcf.gz

# Learn FFPE / OxoG orientation bias
gatk LearnReadOrientationModel -I f1r2.tar.gz -O orientation.tar.gz
# Estimate cross-sample contamination
gatk GetPileupSummaries \
    -I tumor.bam \
    -V af-only-gnomad-common-biallelic.vcf.gz \
    -L common-biallelic.bed \
    -O tumor.pileups.table

gatk CalculateContamination \
    -I tumor.pileups.table \
    -matched normal.pileups.table \
    -O contamination.table \
    --tumor-segmentation segments.table
# Apply all filters (orientation + contamination + germline)
gatk FilterMutectCalls \
    -R reference.fa \
    -V unfiltered.vcf.gz \
    --contamination-table contamination.table \
    --tumor-segmentation segments.table \
    --ob-priors orientation.tar.gz \
    --stats unfiltered.vcf.gz.stats \
    -O filtered.vcf.gz

# Keep only PASS variants
bcftools view -f PASS filtered.vcf.gz -O z -o somatic.PASS.vcf.gz
