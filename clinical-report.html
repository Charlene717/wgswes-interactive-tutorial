臨床報告與資料庫

ClinVar、gnomAD v4、ACMG SF v3.2、HGVS 命名與報告結構：從變異到可作為臨床決策依據。

ClinVar, gnomAD v4, ACMG SF v3.2, HGVS nomenclature, and report structure — from variant to clinically actionable.

概覽

臨床報告的「五個一定要」

NGS 臨床報告不只是輸出 VCF：它是醫療紀錄。需符合 CAP/CLIA / ISO 15189 實驗室規範、ACMG/AMP 分類、HGVS 命名，並提供：

  • 檢驗適應症 (clinical indication) — 為何做此檢測
  • 方法與限制 — 覆蓋範圍、靈敏度、無法偵測的變異類型
  • 發現的變異 — P/LP/VUS（B/LB 通常不報）含 HGVS、ClinVar、gnomAD、ACMG 依據
  • 不確定性聲明 — VUS 解釋、reanalysis policy
  • 建議 — 家族檢測、再分析時機、後續追蹤

An NGS clinical report is not a VCF dump — it is a medical record. Must meet CAP/CLIA / ISO 15189 standards, follow ACMG/AMP, use HGVS naming, and include:

  • Clinical indication — why testing was ordered
  • Method & limits — coverage, sensitivity, undetectable variant types
  • Findings — P/LP/VUS (B/LB usually not reported) with HGVS, ClinVar, gnomAD, ACMG basis
  • Uncertainty statement — VUS interpretation, reanalysis policy
  • Recommendations — family testing, reanalysis timing, follow-up
💡
HGVS 命名（Den Dunnen 2016, Hum Mutat）：編碼 DNA 用 c.（如 c.5266dup），蛋白用 p.（如 p.Lys41Glu 或單碼 p.K41E）。基因組座標用 g.，前綴必加 transcript ID + 版本（NM_007294.4:c.5266dup）。 HGVS (Den Dunnen 2016, Hum Mutat): coding DNA uses c. (e.g. c.5266dup); protein uses p. (e.g. p.Lys41Glu or short p.K41E); genome uses g.; always prefix with transcript ID + version (NM_007294.4:c.5266dup).

人群頻率

二、gnomAD v4 與次要發現

gnomAD v4 (Karczewski 2024, Nature)：807,162 exomes + 76,215 genomes，最大族裔多樣性人群基因組頻率資源。用法：(1) MAF 過濾 (BA1/BS1/PM2)；(2) gene constraint metrics（LOEUF、pLI、misZ）；(3) structural variant frequency。

ACMG SF v3.2 (Miller 2023, Genet Med)：定義 81 個次要發現基因（incidental findings），這些基因即使非檢測適應症發現變異，因有「可行動性 actionable」，應通報病人——例如 BRCA1/2、MLH1、MYBPC3、RYR1 等。

gnomAD v4 (Karczewski 2024, Nature): 807,162 exomes + 76,215 genomes — the largest, most ancestry-diverse population AF resource. Uses: (1) MAF filtering (BA1/BS1/PM2); (2) gene constraint metrics (LOEUF, pLI, misZ); (3) SV frequency.

ACMG SF v3.2 (Miller 2023, Genet Med): defines 81 secondary-findings genes — variants in these are actionable and should be reported even outside the test indication. Examples: BRCA1/2, MLH1, MYBPC3, RYR1.

Q3:

Gene in ACMG SF v3.2 (81 genes)? → Enter SF review.
Q2:
Variant P or LP? → Report only if patient opted-in on consent.
Q3:
